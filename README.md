# harvard-cs50-intro-python
Design and implementation of web apps with Python, JavaScript, SQL and frameworks like Django, React, and Bootstrap. Topics database design, scalability, security, and user experience. Learn how to write and use APIs, create interactive UIs, and leverage cloud services like GitHub and Heroku.  Learn to read, write code, how to test and “debug” it.
